# AI Interview Intelligence Platform

An AI-powered platform that analyzes mock interview responses using speech-to-text, NLP evaluation, and progress analytics.

## Goal

Users can upload or record interview answers and receive structured feedback on clarity, relevance, confidence, and improvement areas.

## Tech Stack

- Frontend: React + Tailwind CSS
- Backend: FastAPI
- Database: PostgreSQL
- AI/NLP: Whisper, embeddings, FAISS
- Async Processing: Celery + Redis
- Deployment: Docker, Vercel, Render/Railway
